# torygg
A library for managing mods for Skyrim Special Edition on linux

WIP.

Requires in path:
- [p7zip](https://github.com/p7zip-project/p7zip) for extracting archives

## Features
- Install / Uninstall mods
- FOMOD support (largely untested)
- Profiles
